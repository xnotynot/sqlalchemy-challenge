Weather Data - Hawaii, Honolulu

Welcome to the Hawaii, Honolulu Weather Data API Hawaii! Use the following API guidelines to access the appropriate data

API Documentation

URL to access the APIs http://localhost:5009 to get the appropriate dataset. All responses are in JSON format.

Route URL Description
Precipitation /api/v1.0/precipitation Returns Precipitation Value for last 12 months grouped by date.
URL:
/api/v1.0/precipitation
Stations /api/v1.0/stations Returns list of stations from the dataset.
URL:
/api/v1.0/stations
Temperature /api/v1.0/tobs Temperature observations from a year from the last data point.
URL:
/api/v1.0/tobs
Temperature Calculations /api/v1.0/Start Date
/api/v1.0/Start Date/End Date		 Returns a list average temperature, minimum temperature, and maximum temperature for a given start or start-end range. Expected Date formatYear-Month-Day (YYYY-MM-DD) format.

/api/v1.0/Start Date
/api/v1.0/2016-01-08
Returns the temperature min,ave and max for the time period greater than, or equal to 2010-01-08.

/api/v1.0/Start Date/End Date
When given the start and the end date, Returns the average, minmum and maximum temperatures between the start and end date
For Example -
/api/v1.0/2016-01-01/2016-01-08
Returns the temperature min,ave and max for a given period, including the two dates.
